# Voice Assistant using Python

## Project Description
This project is a Python-based Voice Assistant developed as part of an internship assignment.  
The assistant listens to voice commands, processes them, and responds using speech synthesis.

---

## Features
- Voice command recognition
- Text-to-speech response
- Time query support
- Greeting and exit commands

---

## Technologies Used
- Python
- speechrecognition
- pyttsx3

---

## How It Works
1. Captures voice input using microphone
2. Converts speech to text
3. Processes command using conditional logic
4. Responds using text-to-speech

---

## How to Run
1. Install dependencies:
   ```bash
   pip install SpeechRecognition pyttsx3 pyaudio
   
2. Run the assistant:
   ```bash
   python voice_assistant.py
